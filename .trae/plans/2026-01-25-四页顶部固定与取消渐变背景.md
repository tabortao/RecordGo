# 2026-01-25 四页顶部固定与取消渐变背景

## 目标

- 任务/心愿/作业家/我的：页面顶部区域固定在最上方，中间内容区域独立滚动。
- 创建心愿页、创建任务页：取消页面渐变背景（保持简洁纯色背景与磨砂卡片风格）。

## 范围

- 前端：App 级布局承载（避免把 padding/class 注入到页面根节点影响固定布局）
- 前端：TasksPage / WishesPage / HomeworkPage / MinePage 布局结构
- 前端：SettingsShell（创建页关闭装饰背景）
- 前端：TaskCreatePage Tab 激活态背景（去渐变）

## 方案

- App：用一个 `div` 包裹路由视图并承接底部留白，让页面根节点不再被自动注入 `pb-24`，便于做“顶部固定 + 中间滚动”。
- 四个主页面：统一为 `flex-col + overflow-hidden`，顶部 header 放在 `shrink-0`，内容区使用 `flex-1 overflow-y-auto` 实现独立滚动。
- 创建页背景：SettingsShell 增加 `decor` 开关；创建心愿/创建任务页面关闭装饰层；创建任务页 Tab 激活态背景改为纯色。

## 验证点

- 四个主页面：顶部卡片不随内容滚动；内容区可滚动且不出现整页双滚动条。
- 下拉刷新（任务页）：仅在内容区滚动到顶部时可触发，下拉动画与刷新提示正常。
- 创建心愿/创建任务：背景无渐变/光晕装饰，整体仍保持磨砂卡片风格与可读性。
- 前端检查：`pnpm run lint; pnpm run lint:fix; pnpm run typecheck` 通过（允许既有 warning）。

- [x] 已完成并自测通过
  - 四个主页面：顶部固定 + 内容区独立滚动
  - 创建心愿/创建任务：背景装饰关闭，Tab 激活态无渐变
