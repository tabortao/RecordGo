# 2026-01-25 任务与心愿页简洁背景 · 导航磨砂 · 统计页美化

## 目标
- 任务页面：背景改为纯色/简洁（不使用渐变背景），保留卡片与顶栏的玻璃层级。
- 底部导航栏：保留磨砂玻璃效果，但更克制（弱化彩色渐变与强光）。
- 任务统计页面：统一到与任务页一致的玻璃风格，信息层级更清晰。
- 心愿页面：取消背景渐变色，整体更简洁、对比更舒适。

## 范围
- 任务页：`TasksPage.vue`
- 底部导航：`BottomNav.vue`（与 `App.vue` 的底部留白联动）
- 任务统计页：`TasksStatsPage.vue`
- 心愿页：`WishesPage.vue`（或心愿模块对应页面）

## 关键设计策略
- 简洁背景：使用纯色（浅色灰/深色灰）+ 轻微纹理/阴影层级（不引入大面积渐变）。
- 磨砂玻璃：`backdrop-filter: blur(...)` + 半透明底色 + 细边框 + 柔阴影；暗色模式提升对比度。
- 信息层级：标题/副标题/指标卡/图表分区明确，减少“满屏彩色块”的干扰。

## 实施步骤
1. 任务页移除背景渐变/柔光层，仅保留顶栏与卡片玻璃效果。
2. 底部导航将彩色背景降噪：去掉强彩 radial，保留磨砂与激活态轻提示。
3. 任务统计页替换渐变背景，改为简洁背景 + 玻璃卡分区 + 更清晰的图表容器。
4. 心愿页去除背景渐变，统一为简洁背景与玻璃卡片体系（保留内容可读性）。
5. `pnpm run typecheck` 与诊断检查，确保无回归。

## 验收标准
- 任务页与心愿页背景不再出现大面积渐变色，整体更克制。
- 底部导航栏有明显磨砂玻璃质感，但不会“彩色漂浮”干扰内容。
- 任务统计页信息层级清晰、视觉一致；浅色/深色均可读。
- typecheck 通过，编辑器无新增诊断错误。
