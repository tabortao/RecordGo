# 2026-01-26 任务备注与多页面 UI 修复

## 目标

- 美化「任务备注」页面，提升阅读与编辑体验（移动端优先）。
- 修复后台管理页移动端表头布局异常。
- 小成长：去掉记录卡片标签前的 `#`。
- 朗读设置：默认选中「自定义接口」下的「云扬」。
- 课表：深色主题下移除渐变底色。
- 修复「生长记录 → BMI 指标说明 → 返回」后再返回的导航栈错误。
- 清理/移除不需要的 `svg` 图标或引用。

## 范围与约束

- 技术栈：Vue 3 + Vite + TypeScript + Element Plus + Pinia。
- 不执行 `pnpm build`；使用热更新验证即可。
- 文案与说明使用中文。

## 执行清单（逐步消项）

- [ ] 任务备注页：信息层级、间距、输入态、空状态与按钮样式统一
- [ ] 后台管理页：移动端表头可横向滚动/自动换行且不遮挡
- [ ] 小成长页：标签渲染逻辑去掉 `#`
- [ ] 朗读设置：默认值改为「自定义接口-云扬」
- [ ] 课表页：深色主题移除渐变背景
- [ ] 导航问题：返回行为按预期回到「小成长」而非 BMI 说明
- [ ] 移除 svg：删除不再使用的图标或引用，避免冗余
- [ ] 验证：lint + typecheck 通过，手动回归关键链路

## 验收标准

- 任务备注页在 375px 宽度下无溢出/遮挡；输入与展示状态切换自然。
- 后台管理表格在移动端表头与列对齐，左右滚动正常。
- 小成长记录卡片标签不再显示 `#` 前缀。
- 朗读设置首次进入默认选中「云扬」且可正常保存/切换。
- 课表深色主题背景为纯色/透明，无渐变层叠。
- 导航链路：生长记录 → BMI 指标说明 → 返回 → 生长记录 → 左上返回 → 小成长。
