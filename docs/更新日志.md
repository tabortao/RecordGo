## v0.3.0

发布时间：2025-11-27

### 🐛 修复

refactor(storage): 统一用户文件存储路径结构

修改所有用户上传文件的存储路径，将原路径中的资源类型目录（如 avatars、wish 等）移动到用户 ID 目录下，形成统一结构：`images/{user_id}/{resource_type}`。同时更新相关文档说明此变更。

- 头像文件路径改为 uploads/images/{user_id}/avatars
- 任务图片路径改为 uploads/images/{user_id}/task_images
- 心愿图片路径改为 uploads/images/{user_id}/wish
- 音频附件路径改为 uploads/images/{user_id}/task_images
