在 RecordGo 项目中，底部导航栏增加“作业家”按钮，点击进入作业家页面，页面可以分类展示各种作业相关功能，按科目进行分类，现在我要在语文分类下方，有一个拼音的页面，点击图标进入拼音页面，拼音页面后续会包含很多拼音相关的功能，现在在拼音左上角新增一个互动游戏——拼音大转盘。每个页面都有一个对应的图标，点击图标后进入对应的页面。 每个页面左上角都有一个返回图标，点击返回上一个页面。

请模仿如上图的拼音转盘（外圈声母、中圈韵母及韵母组合、内圈四个声调）做一个拼音大转盘的互动游戏，需要响应式设计，适配手机 / 平板，按钮大小适合儿童点击，并在页面的下方对应显示出转盘停止后箭头指的拼音组合，比如：b+a+一声=bā，并朗读对应的发音。注意实际不存在的拼音组合，在游戏里不要生成。

交互功能：
点击 “转动” 按钮，转盘随机旋转后停下，自动匹配声母 + 韵母，拼音右侧展示对应汉字的词组，如 bà | 爸爸。
若声母 + 韵母无法拼读（如 f+i），自动过滤掉，自动匹配下一个组合，ts库中构建符合要求的拼音组合。
转动结束后，不自动发音，点击“朗读发音”按钮，播放对应声调的词组的发音。
视觉与体验：
风格：卡通可爱的浅绿色系，图标生动（适配 3-8 岁儿童）。
音效：转动时播放轻快音效，拼读正确时播放 “叮” 的提示音。
响应式：适配手机 / 平板，按钮大小适合儿童点击。
