# 古诗词需求文档

## 1. 应用概述

### 1.1 应用名称
古诗词打卡

### 1.2 应用描述
一款面向学前教育、小学生、中学生的古诗词学习打卡APP，通过背诵打卡、默写打卡、每日一诗、闯关挑战等多种形式，帮助用户养成古诗词学习习惯，提升诗词理解和记忆能力。
### 1.3 设计风格
采用马卡龙风格，色彩柔和温馨，营造轻松愉悦的学习氛围

## 2. 核心功能

### 2.1 古诗词库
- 内置小学、初中、高中课本必备古诗词（需补全搜索并添加）
- 支持用户新增、编辑、删除古诗词卡片
- 编辑古诗词卡片时可上传音频链接，在卡片中展示播放图标，点击即可播放
- 古诗词卡片原文Tab包含：古诗标题（title_cns）、作者（author_cns）、作者朝代（dynasty_cns）、古诗内容（paragraphs_cns）、音频链接（编辑时输入音频链接，古诗卡片原文需展示播放图标，点击可以播放对应音频）、图片链接（编辑时输入图片链接，古诗卡片原文古诗内容背景显示该图片）
- 古诗词卡片原文可展示或隐藏拼音
- 古诗词卡片赏析Tabs包含：
  - 译文：展示古诗译文
  - 赏析：展示古诗赏析
  - 趣事：展示古诗相关趣事
  - 点击“赏析”Tab后，自动调用AI生成本古诗的译文、赏析、趣事，并保存到用户本地。
- 古诗词库分类：
  - 小学诗卡：展示所有小学阶段古诗卡片
  - 初中诗卡：展示所有初中阶段古诗卡片
  - 高中诗卡：展示所有高中阶段古诗卡片
- 古诗词卡片右上角展示：
  - 该古诗学习次数
  - 掌握状态（不同底纹区分：已掌握/未掌握）
- 古诗词库管理 
  - 支持添加自定义古诗词
    - 填写诗词名称、作者、朝代、内容等基本信息
    - 选择学段（小学/初中/高中）
    - 可选填译文、赏析、趣事等详细信息
    - 支持上传音频链接
  - **编辑功能**：可以编辑已有的古诗词信息
  - **删除功能**：支持删除不需要的古诗词（带确认对话框），仅可删除用户自定义的古诗词，不可删除系统古诗词。
  - **音频标识**：有音频的古诗词会显示🎵标识
### 2.2 学习计划
- 首页展示学习计划
- 展示已背诵古诗数量
- 展示未背诵古诗数量
- 根据艾宾浩斯记忆曲线安排每日学习和复习内容
- 显示当日需学习的古诗
- 显示当日需复习的古诗

### 2.3 打卡日历
- 打卡日历支持查看和筛选所有日期，要紧凑显示，默认显示最近的周一到周日日期、周几，点击日期可查看每日学习情况。
- 日期展示格式：显示该日打卡的古诗数量（新学数量 + 复习数量）
- 点击日期可查看该日学习的所有古诗详情

### 2.4 背诵打卡
- 用户可选择诗词进行背诵打卡
- 支持TTS语音功能进行古诗朗读
- 支持语音跟读功能，系统或AI可判断背诵准确度
- 支持功能：全部展示、全部隐藏、局部隐藏、首字提示
- 完成背诵后可标记打卡，系统记录背诵历史
- 支持查看背诵打卡天数和连续打卡记录

### 2.5 默写打卡
- 用户可选择诗词进行默写打卡
- 支持文字输入默写，系统自动批改
- 完成默写后可提交，系统记录默写结果
- 支持查看默写正确率和错字统计

### 2.6 每日一诗
- 每日推荐一首经典古诗词
- 展示诗词原文Tab、赏析Tab
- 支持收藏功能，方便用户随时查看

### 2.7 闯关挑战
- 设置不同难度等级的诗词闯关关卡
- 闯关形式包括选择题、填空题、排序题等
- 完成关卡可获得积分和徽章奖励
- 支持查看闯关进度和排行榜

### 2.8 打卡记录分享
- 用户可将每日打卡完成情况生成海报或卡片
- 支持分享到微信、朋友圈等社交平台
- 分享内容包含打卡天数、完成情况、学习成果等

### 2.9 个人中心
- 展示用户基本信息和学习数据
- 显示累计打卡天数、背诵诗词数量、默写正确率等统计数据
- 支持查看打卡记录、收藏诗词、历史成绩等

### 2.10 学习计划
- 基于艾宾浩斯记忆曲线的智能复习提醒
- 显示今日待学习和待复习的诗词
- 学习进度可视化
- 打卡日历展示

## 3. 用户群体

### 3.1 学前教育
- 通过趣味化的诗词内容和语音跟读，培养幼儿对古诗词的兴趣
- 诗词难度适中，以简单易懂的童谣和经典诗句为主

### 3.2 小学生
- 涵盖小学语文课本中的所有古诗词内容
- 提供注释、译文和赏析，帮助学生理解诗词含义
- 通过打卡和闯关形式，巩固课堂所学知识

### 3.3 中学生
- 涵盖初中和高中语文课本中的古诗词内容
- 提供更深入的诗词赏析和创作背景介绍
- 通过闯关挑战和默写练习，提升诗词理解和应用能力

## 4. 技术说明
- 古诗词数据库使用chinese_poems.json，需要正确理解其格式方便正确调用。
- 联网搜索小学生、初中生、高中生必备古诗词，使用单独的json文件对chinese_poems.json中的古诗进行分类标记。
- 代码质量：通过ESLint检查，代码规范统一
- 类型安全：完善TypeScript类型定义
- 用户体验：优化交互流程，添加加载状态和错误提示

## 5. 设计特色

- **马卡龙风格**：采用柔和温馨的粉紫色、淡蓝色、淡绿色配色
- **响应式设计**：完美适配手机、平板、电脑
- **动画效果**：流畅的页面切换和交互动画

## 6. 功能亮点

### 艾宾浩斯记忆曲线
应用内置艾宾浩斯记忆曲线算法，自动安排复习计划：
- 首次学习后1天复习
- 第2天复习
- 第4天复习
- 第7天复习
- 第15天复习
- 第30天复习

### 智能语音识别
- 支持语音跟读背诵
- 自动识别背诵内容
- 计算背诵准确度
- 提供详细的识别结果

### 默写智能批改
- 自动对比原文
- 标注错误字词
- 显示漏写和多写
- 计算正确率

### 打卡激励系统
- 连续打卡天数统计
- 打卡日历可视化
- 学习成就展示
- 学习时长记录



## 7. 数据说明

- 个人学习进度、学习记录、复习计划、闯关进度、统计数据、自定义古诗词等数据，需要保存到后台数据库中。
